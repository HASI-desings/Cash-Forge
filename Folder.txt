Root Directory: `cashforge-gold/`
├── README.md                  # Project overview, setup (copy .env.example, Tailwind CDN, Supabase project link), blueprint summary, dev commands.
├── .gitignore                 # Standard: node_modules/, .env, dist/, *.log.
├── package.json               # Optional: Dev deps (esbuild for bundling, vitest for tests); scripts: "dev", "build".
├── index.html                 # Main SPA shell (loads router; <head> with Tailwind CDN, meta for PWA). Redirects unauth to auth.html.
├── router.html                # Fallback for direct page access (JS routes to SPA shell).
├── config/                    # Supabase-only env (no other backend).
│   ├── .env.example           # SUPABASE_URL=your_project_url; SUPABASE_ANON_KEY=your_anon_key (gitignore .env).
│   └── supabase.js            # Client init: import { createClient } from 'https://esm.sh/@supabase/supabase-js'; export const supabase = createClient(URL, ANON_KEY).
├── pages/                     # HTML templates (loaded via router; dynamic content via JS components).
│   ├── auth.html              # Register/Login (liquid slider, ?ref=XYZ prefill; Supabase auth.signUp/signIn).
│   ├── home.html              # Dashboard (carousel, package accordion; upgrade calc client-side).
│   ├── teams.html             # Affiliates (A/B/C tabs; VIP progress from Supabase query).
│   ├── deposit.html           # Deposit (USDT/PKR calc; screenshot upload to Supabase Storage).
│   ├── withdraw.html          # Withdraw (presets, PIN keypad; weekend lock client-check, Supabase upsert).
│   └── tasks.html             # "Work" (3 tasks, timers; anti-cheat via Visibility API, claim via Supabase).
├── components/                # Reusable UI (HTML/JS; dynamic import in router).
│   ├── header.js              # Fixed header (balance pill; listens to CustomEvent from state.js).
│   ├── nav.js                 # Bottom nav (active bubble; haptic on tap).
│   ├── carousel.js            # Hero slider (lazy-load via IntersectionObserver).
│   ├── accordion.js           # Packages (diff calc: target investment - current).
│   ├── tabs.js                # Teams (filter A/B/C via Supabase relations query).
│   ├── modal.js               # Modals (PIN, toasts; optimistic updates).
│   ├── keypad.js              # PIN input (6-digit, client-hash before Supabase).
│   ├── progress.js            # VIP bars (salary preview from config).
│   └── task-timer.js          # Tasks (random income split; tab-switch detect fails task).
├── assets/                    # Static files.
│   ├── js/                    # ES6 modules (import from supabase.js for DB calls).
│   │   ├── config.js          # CONSTS: Packages[], VIPLevels[], Referrals (fixed/%/daily), ExchangeRate=285, WithdrawPresets[], ProTradingTiers[] (3 + placeholders).
│   │   ├── state.js           # Global state (setBalance/updateBalance dispatches 'balance-updated'; optimistic UI pre-DB).
│   │   ├── db.js              # Client-side Supabase wrappers: createProfile (gen ref_code, insert user w/ referred_by), calculateCommissions (client lookup + Supabase RPC for uplines), checkVipStatus (query subs, filter Basic, update vip_level).
│   │   ├── utils/             # Helpers (all client-side for bug-proofing).
│   │   │   ├── math.js        # PKR precision (Number.toFixed(2) or BigInt for commissions/ROI).
│   │   │   ├── referral.js    # 3-tier logic: invitation bonus table lookup, % by range (15/5/2 etc.), daily 5/2/1%.
│   │   │   ├── vip.js         # Count subs (A-only or A+B+C, exclude Basic), salary sim (monthly 5th trigger via cron? Client preview only).
│   │   │   ├── trading.js     # Pro Tiers (Day/Weekly/Monthly ROI; placeholders; locked timer client-display).
│   │   │   ├── validation.js  # Sanitize (PIN, ref_code, weekend: new Date().getDay() >0 && <6 lock).
│   │   │   └── haptic.js      # if (navigator.vibrate) vibrate(50) on success.
│   │   ├── router.js          # Hash routing (#/home -> import/load components; Supabase auth check on route).
│   │   └── events.js          # CustomEvents: balance-updated, task-complete, commission-trigger.
│   ├── css/                   # Styles.
│   │   ├── tailwind.css       # @tailwind base/components/utilities; (CDN in HTML).
│   │   └── style.css          # Glassmorphism: .glass { backdrop-filter: blur(10px); background: rgba(255,255,255,0.1); }, anims (keyframes slideUp, pulse-glow, shake), colors (cyan #00FFFF, slate #1e293b).
│   ├── images/                # WebP/SVG (lazy-load).
